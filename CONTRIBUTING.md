# Contributing to janNet

Thanks for your interest in contributing! This project is in active development and we welcome all contributions.

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/janNet.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Push and open a PR

## Development Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run
python main.py
```

2. Update README if you add new features
3. Test your changes locally
4. Describe what you changed and why
